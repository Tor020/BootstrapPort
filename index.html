About Me

At time of writing in 2017, I'm a 26 year old self taught Web Developer whose been using the Internet since the late 90s. I've always thought the Internet was pretty awesome, but I especially like the practical application it brings to everyone.

The first web application I made and put on github for use is called the KF2calc which was made to help players to calculate their damage in an online video game. Damage calculations weren't displayed anywhere in-game or online and you'd have to look up formulas on the Internet and plug them into a calculator for each weapon. After doing it myself a few times I thought that it'd be pretty useful to have an application that did this for me. So I went to building the website.

I launched the website last year and have implemented additional features after communicating with the userbase on social media. Since then, I've monetized the website with Ad Sense and have daily usage confirmed by Google Analytics. Throughout this project's lifetime I've tried to practice Agile and Waterfall software development methodologies.

I'm really dedicated to mastering Javascript and working to be able to utilize it to its full potential. This includes things like learning the nuances of modern Javascript frameworks. From there I want to branch out into backend languages like python and php and eventually become a fullstack developer.

Technical Skills

Javascript - JQuery - CSS3 - HTML5 - Bootstrap - Git - Gulp
